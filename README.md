# How to run the app

Clone the project:

`git clone https://github.com/jacquelineb/code-the-dream-prework.git`

Move into project directory:

`cd code-the-dream-prework`

Ensure Node.js is installed on your computer and then install dependencies:

`npm install`

Next, start the server:

`npm start`

This will open the homepage of the webapp in a browser @ `localhost:3000`.
